Livestock Management System

Overview
The Livestock Management System is designed to help farmers or organizations track livestock, manage records, and monitor key data such as breeds, feeding, health, and growth.

Features
- Record and manage livestock data
- Categorization by breed and type
- Health monitoring
- Feeding schedules
- Farm performance metrics
- Admin dashboard

Tech Stack
- **Frontend:** HTML, CSS, JavaScript
- **Backend:** PHP
- **Database:** MySQL
- **Additional Tools:** XAMPP / WAMP for local server

How to Run
1. Import the SQL database into phpMyAdmin
2. Place project folder inside `htdocs`
3. Run Apache + MySQL
4. Access via: `http://localhost/livestock-mngt`
